### Factor ComĂşn

</TopicSubsection>

<TopicSubsection>

#### Procedimiento

<TopicSubsectionContent>

1. Identificar el factor que se repite en todos los tĂŠrminos.
2. Extraer el mĂĄximo comĂşn divisor (MCD).
3. Dividir cada tĂŠrmino por el factor comĂşn.

</TopicSubsectionContent>

</TopicSubsection>

<TopicSubsection>

#### Ejemplos

<TopicSubsectionContent>

| ExpresiĂłn | Factor ComĂşn | FactorizaciĂłn |
| --- | --- | --- |
| $6x + 9$ | $3$ | $3(2x + 3)$ |
| $x^2 + x$ | $x$ | $x(x + 1)$ |
| $4x^3 - 8x^2$ | $4x^2$ | $4x^2(x - 2)$ |
| $6x^2y + 9xy^2$ | $3xy$ | $3xy(2x + 3y)$ |

</TopicSubsectionContent>

</TopicSubsection>

<TopicSubsection>

### Factor ComĂşn por Grupos

<TopicSubsectionContent>

TambiĂŠn llamado agrupaciĂłn de tĂŠrminos. Se usa cuando no hay factor comĂşn global, pero sĂ­ por bloques.

**Ejemplo general:**

$$
ax + bx + ay + by
$$

1. Agrupar: $(ax + bx) + (ay + by)$
2. Factor comĂşn por grupo: $x(a + b) + y(a + b)$
3. Factor comĂşn final: $(a + b)(x + y)$

</TopicSubsectionContent>

</TopicSubsection>

### Diferencia de Cuadrados

</TopicSubsection>

<TopicSubsection>

#### FĂłrmula

<TopicSubsectionContent>

$$
a^2 - b^2 = (a - b)(a + b)
$$

**Reconocimiento:** Dos tĂŠrminos, ambos cuadrados perfectos, separados por un signo menos.

</TopicSubsectionContent>

</TopicSubsection>

<TopicSubsection>

#### Ejemplos

<TopicSubsectionContent>

| ExpresiĂłn | IdentificaciĂłn | FactorizaciĂłn |
| --- | --- | --- |
| $x^2 - 25$ | $a=x$, $b=5$ | $(x - 5)(x + 5)$ |
| $9x^2 - 16$ | $a=3x$, $b=4$ | $(3x - 4)(3x + 4)$ |
| $x^4 - 1$ | $a=x^2$, $b=1$ | $(x^2 - 1)(x^2 + 1) = (x-1)(x+1)(x^2+1)$ |
| $\frac&#123;1&#125;&#123;4&#125; - x^2$ | $a=\frac&#123;1&#125;&#123;2&#125;$, $b=x$ | $\left(\frac&#123;1&#125;&#123;2&#125; - x\right)\left(\frac&#123;1&#125;&#123;2&#125; + x\right)$ |

</TopicSubsectionContent>

### Trinomio Cuadrado Perfecto

</TopicSubsection>

<TopicSubsection>

#### Reconocimiento

<TopicSubsectionContent>

Un trinomio $a^2 \pm 2ab + b^2$ es cuadrado perfecto si:

- El primer y tercer tĂŠrmino son cuadrados perfectos.
- El tĂŠrmino del medio es $\pm 2ab$ (el doble producto de las raĂ­ces).

</TopicSubsectionContent>

</TopicSubsection>

<TopicSubsection>

#### FĂłrmulas

<TopicSubsectionContent>

$$
a^2 + 2ab + b^2 = (a + b)^2
$$

$$
a^2 - 2ab + b^2 = (a - b)^2
$$

</TopicSubsectionContent>

</TopicSubsection>

<TopicSubsection>

#### Ejemplos

<TopicSubsectionContent>

| ExpresiĂłn | VerificaciĂłn | FactorizaciĂłn |
| --- | --- | --- |
| $x^2 + 6x + 9$ | $2ab = 2(x)(3) = 6x$ | $(x + 3)^2$ |
| $x^2 - 10x + 25$ | $2ab = 2(x)(5) = 10x$ | $(x - 5)^2$ |
| $4x^2 + 12x + 9$ | $2ab = 2(2x)(3) = 12x$ | $(2x + 3)^2$ |
| $9x^2 - 6x + 1$ | $2ab = 2(3x)(1) = 6x$ | $(3x - 1)^2$ |

</TopicSubsectionContent>

</TopicSubsection>

<TopicSubsection>

### Trinomio de la Forma $x^2 + bx + c$

</TopicSubsection>

<TopicSubsection>

#### Procedimiento

<TopicSubsectionContent>

Buscar dos nĂşmeros $r_1$ y $r_2$ tales que:

- $r_1 + r_2 = b$ (suma = coeficiente de $x$)
- $r_1 \cdot r_2 = c$ (producto = tĂŠrmino independiente)

**FactorizaciĂłn:** $x^2 + bx + c = (x + r_1)(x + r_2)$

</TopicSubsectionContent>

</TopicSubsection>

<TopicSubsection>

#### Ejemplos

<TopicSubsectionContent>

| ExpresiĂłn | Buscar nĂşmeros | FactorizaciĂłn |
| --- | --- | --- |
| $x^2 + 5x + 6$ | $2 + 3 = 5$ y $2 \times 3 = 6$ | $(x + 2)(x + 3)$ |
| $x^2 - 5x + 6$ | $(-2) + (-3) = -5$ y $(-2) \times (-3) = 6$ | $(x - 2)(x - 3)$ |
| $x^2 + x - 6$ | $3 + (-2) = 1$ y $3 \times (-2) = -6$ | $(x + 3)(x - 2)$ |
| $x^2 - 7x + 12$ | $(-3) + (-4) = -7$ y $(-3) \times (-4) = 12$ | $(x - 3)(x - 4)$ |

</TopicSubsectionContent>

</TopicSubsection>

<TopicSubsection>

### Trinomio de la Forma $ax^2 + bx + c$ (MĂŠtodo AC)

</TopicSubsection>

<TopicSubsection>

#### Procedimiento

<TopicSubsectionContent>

1. Calcular $A \times C$ (multiplicar primer y Ăşltimo coeficiente).
2. Buscar dos nĂşmeros que sumen $b$ y multipliquen $AC$.
3. Reescribir el tĂŠrmino medio usando esos nĂşmeros.
4. Factorizar por agrupaciĂłn.

</TopicSubsectionContent>

</TopicSubsection>

<TopicSubsection>

#### Ejemplo

<TopicSubsectionContent>

$3x^2 + 10x + 8$

1. $AC = 3 \times 8 = 24$
2. Buscar dos nĂşmeros que sumen $10$ y multipliquen $24$: $4$ y $6$
3. Reescribir: $3x^2 + 4x + 6x + 8$
4. Agrupar: $(3x^2 + 4x) + (6x + 8) = x(3x + 4) + 2(3x + 4)$
5. Factor comĂşn: $(3x + 4)(x + 2)$

</TopicSubsectionContent>

</TopicSubsection>

<TopicSubsection>

### Suma y Diferencia de Cubos

</TopicSubsection>

<TopicSubsection>

#### FĂłrmulas

<TopicSubsectionContent>

$$
a^3 + b^3 = (a + b)(a^2 - ab + b^2)
$$

$$
a^3 - b^3 = (a - b)(a^2 + ab + b^2)
$$

**Mnemotecnia:**

- Suma de cubos: $(+)$ binomio, $(-, +)$ trinomio.
- Diferencia de cubos: $(-)$ binomio, $(+, +)$ trinomio.

</TopicSubsectionContent>

</TopicSubsection>

<TopicSubsection>

#### Ejemplos

<TopicSubsectionContent>

| ExpresiĂłn | IdentificaciĂłn | FactorizaciĂłn |
| --- | --- | --- |
| $x^3 + 8$ | $a=x$, $b=2$ | $(x + 2)(x^2 - 2x + 4)$ |
| $x^3 - 27$ | $a=x$, $b=3$ | $(x - 3)(x^2 + 3x + 9)$ |
| $8x^3 + 125$ | $a=2x$, $b=5$ | $(2x + 5)(4x^2 - 10x + 25)$ |
| $27x^3 - 64$ | $a=3x$, $b=4$ | $(3x - 4)(9x^2 + 12x + 16)$ |

</TopicSubsectionContent>

</TopicSubsection>

<TopicSubsectionContent>

| Valor de $\Delta$ | NĂşmero de RaĂ­ces Reales | Tipo de RaĂ­ces |
| --- | --- | --- |

</TopicSubsection>

<TopicSubsection>

<TopicSubsectionContent>

### MĂŠtodo de Ruffini

<TopicSubsectionContent>

**Uso:** Dividir polinomios por $(x - a)$ para factorizar.

</TopicSubsectionContent>

<TopicSubsection>

#### Procedimiento

<TopicSubsectionContent>

1. Identificar posibles raĂ­ces (divisores del tĂŠrmino independiente).
2. Probar cada raĂ­z con Ruffini.
3. Si el resto es $0$, se encontrĂł un factor $(x - a)$.
4. El cociente es el otro factor.

</TopicSubsectionContent>

</TopicSubsection>

<TopicSubsection>

#### Ejemplo

<TopicSubsectionContent>

Probar $x=1$:

```text

  |      1  -5   6
  |________________
     1  -5   6   0
```

FactorizaciĂłn: $(x - 1)(x^2 - 5x + 6) = (x-1)(x-2)(x-3)$
